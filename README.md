# AEDS I - Linguagem C

## Descrição

Este repositório contém exercícios e projetos desenvolvidos em Linguagem C durante a disciplina de Algoritmos e Estruturas de Dados I (AEDS I), do curso de Ciência da Computação.

## Tecnologias Utilizadas

* Linguagem C;
* GCC;
* Visual Studio Code.

## Funcionalidades

* Implementação de algoritmos;
* Uso de funções e recursão;
* Manipulação de vetores e matrizes;
* Utilização de estruturas ("struct");

## Contexto de Desenvolvimento

Os códigos presentes neste repositório foram desenvolvidos como parte das atividades acadêmicas da disciplina de Algoritmos e Estruturas de Dados I (AEDS I), do curso de Ciência da Computação da PUC Minas.
